# Game of Life

### Get the source code

```sh
git clone https://github.com/predatorray/game-of-life.git
```

### Compile the source code using Maven

```sh
cd game-of-life/
mvn clean compile
```

### Enjoy!

```sh
java -cp target/classes/ me.predatorray.GameOfLife
```
